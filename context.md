## ✅ ماژول‌های ساخته‌شده

---

### ۱. صفحه اصلی / Landing Page

**Navbar:**
- شفاف هنگام بالای صفحه، glassmorphism هنگام اسکرول
- لوگو IronZone.ir سمت راست
- لینک‌های ناوبری + دکمه «ورود به پنل» نارنجی

**Hero:**
- عکس تمام‌صفحه از Unsplash با انیمیشن zoom-out
- badge نارنجی با نقطه چشمک‌زن
- شعار: **Train Hard. Live Strong.** با انیمیشن fadeSlideUp پله‌ای
- دکمه «همین الان شروع کن» + دکمه ghost «خدمات ما»
- scroll indicator با انیمیشن bounce

**Stats (IntersectionObserver + counter animation):**
- ۱۲۵۰+ عضو فعال / ۱۲+ سال تجربه / ۲۸+ مربی متخصص
- انیمیشن شمارش با ease-out cubic

**Services (بخش خدمات):**
- ۳ کارت: بدنسازی 🏋️ / کراسفیت 🔥 / یوگا 🧘
- hover effect: بالا آمدن + خط نارنجی + چرخش آیکون
- دکمه «بیشتر بدان» → باز کردن **Service Modal**

**Service Modal:**
- مودال تمام اطلاعات هر ورزش
- آیکون + عنوان + شعار
- توضیح کامل
- ۶ مزیت کلیدی با bullet نارنجی
- چه کسانی مناسب‌اند (۴ دسته)
- ۳ آمار (جلسات/دقیقه/کالری)
- دکمه «همین الان ثبت‌نام کن» ← میره به لاگین
- بستن با کلیک بیرون یا دکمه ✕

**Coaches (مربیان):**
- ۳ کارت مربی با عکس واقعی از Unsplash
- overlay گرادیانت + badge تخصص (بدنسازی/یوگا/کراسفیت)
- اسم + سمت + آمار مربی (سال تجربه / شاگرد / مدال)

**CTA Banner:**
- بنر دعوت به ثبت‌نام با گلو نارنجی

**Footer:**
- لوگو + توضیح باشگاه
- ۴ دکمه شبکه اجتماعی (📸 ✈️ 💬 ▶️)
- ستون‌های لینک: سریع / خدمات / تماس

**Scroll Animations:**
- همه بخش‌ها با `IntersectionObserver`
- کلاس `reveal` → `visible` هنگام ورود به viewport
- delay پله‌ای روی کارت‌ها (`.delay-1`, `.delay-2`, `.delay-3`)

### ۲. صفحه لاگین / Login Page

**Navbar لاگین:**
- عکس Hero باشگاه پس‌زمینه + overlay گرادیانت
- انیمیشن ورودی: fadeInBg → slideDown navbar → slideUp شعار → slideInRight فرم

**فرم — دو تب:**

**تب ورود:**
- نام کاربری + رمز عبور + نقش کاربری
- جلوگیری از تایپ فارسی در فیلدها + هشدار
- لرزش فرم هنگام ورود اشتباه
- پشتیبانی از کاربران پیش‌فرض و ثبت‌نام‌شده

**تب ثبت‌نام:**
- نام و نام خانوادگی (فارسی)
- نام کاربری (انگلیسی، حداقل ۴ کاراکتر)
- رمز عبور (حداقل ۶ کاراکتر)
- تکرار رمز عبور
- نقش: ورزشکار / مربی
- اعتبارسنجی کامل + بررسی تکراری نبودن نام کاربری
- بعد از ثبت‌نام → auto redirect به تب ورود + پر کردن نام کاربری

**دکمه بازگشت:**
- «← بازگشت به صفحه اصلی» بالای فرم
- hover: رنگ نارنجی + حرکت به چپ

### ۳. داشبورد

**Header:**
- لوگو + نام باشگاه سمت راست
- اطلاعات کاربر (نام + نقش) + badge نقش + دکمه خروج

**Sidebar:**
- منوی ناوبری با آیکون
- آیتم فعال با هایلایت نارنجی

**Overview:**
- ۴ کارت آمار: کل اعضا / ورزشکار / مربی / VIP
- لیست آخرین اعضای ثبت‌نام‌شده

**بخش‌های موجود:**
- افزودن عضو (فرم + validation شماره تلفن + تقویم شمسی)
- لیست اعضا (جستجو + فیلتر نقش + حذف با مودال)
- مدیریت اعضا ۴ تب: ثبت‌نام / پروفایل / تاریخچه / عکس

### ۴. تقویم شمسی (ShamsiPicker)

- ساخته‌شده با `jalaali-js` از CDN
- popup با ناوبری ماه/سال (« ‹ › »)
- ماه‌های فارسی + روزهای هفته شمسی
- هایلایت روز امروز با نارنجی
- استفاده در: افزودن عضو / مدیریت ثبت‌نام / تاریخچه
